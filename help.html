❓ ゲームの登録・管理ヘルプ

Stock Inc. Games ポータルでは、GitHub Issue を使ってゲームを管理しています。
誰でも簡単にゲームの登録・編集・削除ができます。

1 ゲームを登録する

1 Issue 作成ページを開く

下の「ゲームを登録する」ボタンをクリックすると、GitHub の Issue 作成フォームが開きます。

※ GitHub アカウントが必要です（無料で作成できます）

2 フォームに情報を入力する

以下の項目を入力してください:

  • Issue タイトル — ゲーム名（例: 🥦 ブロッコリーシューター）
  • ゲームURL — ゲームの公開URL
  • サムネイル画像URL — サムネイル画像の直リンクURL
  • 紹介テキスト — 1〜3文程度の説明
  • カテゴリ — ドロップダウンから選択
  • タグ（任意） — カンマ区切りでタグを入力
3 Submit new issue をクリック

Issue が作成されると、ポータルサイトに自動的にゲームが表示されます。
反映にはキャッシュの関係で最大5分かかる場合があります。

＋ ゲームを登録する

2 登録内容を修正する

1 対象の Issue を開く

下のリンクから登録済みゲームの Issue 一覧を確認し、修正したいゲームの Issue を開きます。

2 Issue の本文を編集する

Issue 本文の右上にある … メニューから「Edit」を選択し、
ゲームURL、サムネイル、紹介テキスト等を修正して保存してください。

⚠️ 注意: ### ゲームURL や ### カテゴリ などのヘッダー行は変更・削除しないでください。ポータルが正しくデータを読み取れなくなります。
📋 登録ゲーム一覧を見る

3 ゲームを削除（非表示に）する

1 対象の Issue を Close する

ポータルから削除したいゲームの Issue を開き、画面下部の「Close issue」ボタンをクリックします。

Issue を Close するとポータルから非表示になります。再度 Open すれば復活します。

💡 ヒント: Issue は削除されるわけではなく「Close」されるだけなので、いつでも元に戻せます。

4 サムネイル画像について

サムネイル画像のURLを取得する方法はいくつかあります:

  • GitHub Issue にアップロード — 任意の Issue のコメント欄に画像をドラッグ&ドロップし、生成されたURLをコピー
  • リポジトリ内の画像を使用 — https://raw.githubusercontent.com/ユーザー名/リポジトリ/main/画像パス の形式
  • 外部 URL を使用 — 直リンク可能な画像URLならどこでもOK
📐 推奨サイズ: アスペクト比 16:10 の PNG または JPG。ポータルのカードでは 16:10 の領域にクロップされて表示されます。

? よくある質問

Q. GitHub アカウントがないと登録できませんか？

はい、GitHub Issue を使うため、GitHub アカウント（無料）が必要です。

Q. 自分で作ったゲームでなくても登録できますか？

はい！公開されている HTML ゲームであれば、どなたが作成したものでも登録できます。ただし、著作権やライセンスにはご注意ください。

Q. 登録したのにポータルに表示されません

キャッシュの関係で反映まで最大5分かかります。ブラウザをハードリロード（Cmd+Shift+R / Ctrl+Shift+R）してみてください。

Q. Issue のラベルは何のためにありますか？

game ラベルが付いた Issue がポータルに表示されます。テンプレートから作成すると自動で付与されます。
